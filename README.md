
## Sumobot ~ 2 Wheels (Arduino IDE) ★🚘
The program is written using the Arduino IDE to control a two-wheeled sumobot. It uses an ultrasonic sensor to measure distance and infrared sensors to detect edges or direction. Based on the distance detected, the robot moves forward, backward, or adjusts its speed. The infrared sensors help the robot decide whether to turn left, turn right, or avoid going out of bounds. Motor functions such as forward, backward, and turning are controlled through specific pin outputs and speed values, allowing the robot to respond quickly during a sumo match.

### 🎀 Functions 
⬆️ moveForward(speed) – Moves the robot forward

⬇️ moveBackward(speed) – Moves the robot backward

⬅️ turnLeft() – Rotates the robot to the left

➡️ turnRight() – Rotates the robot to the right

## ✨ Getting Started

## 1. Prepare Components

Set up the robot parts (Arduino, motors, ultrasonic sensor, infrared sensors, wires, and power supply).

## 2. Connect the Circuit

Wire all components based on the assigned pins in the code (motors, sensors, and power).

## 3. Open the Code

Open the program in Arduino IDE or you can click this link. https://www.arduino.cc/en/software/

## 4. Upload the Code

Connect the Arduino board to your computer and upload the code.

## 5. Power the Robot

Turn on the power supply to activate the robot.

## 6. Test the Robot 🤖

Place the robot on the arena. It will move, detect distance, and respond using its sensors automatically.
